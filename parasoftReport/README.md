# Parasoft Report Folder

This folder contains Parasoft HTML reports used by the analysis tool.

## 📄 Expected Files

- **report_dev1.html** - Main Parasoft HTML report from static analysis

## 🔧 Usage

### Option 1: Use Existing Parasoft Report
Place your `report_dev1.html` file (generated by Parasoft C/C++test or similar tool) in this folder.

### Option 2: Auto-Generate Report
If `report_dev1.html` is not present, the tool can automatically generate a MISRA/CERT compliance report from source code using the built-in static analyzer.

## 📝 How to Get report_dev1.html

### From Parasoft C/C++test:
1. Run static analysis in Parasoft C/C++test
2. Export results as HTML report
3. Rename to `report_dev1.html`
4. Place in this folder

### From Polyspace:
1. Run Polyspace analysis
2. Export HTML report
3. Rename to `report_dev1.html`
4. Place in this folder

### Auto-Generate (Built-in):
If you don't have Parasoft or Polyspace, the tool will automatically create a report when you provide source code path:

```bash
# GUI: Provide source path in Input Path field
scripts\Launch_GUI.bat

# Batch: Provide source path when prompted
batch_support\Run.bat
```

## 🚀 Quick Start

1. **Place report here:**
   ```
   parasoftReport\report_dev1.html
   ```

2. **Run analysis:**
   ```bash
   scripts\Launch_GUI.bat
   ```

3. **Or use batch files:**
   ```bash
   batch_support\Run.bat
   ```

---

**Note:** This folder is for input reports. Generated analysis results go to the `reports\` folder.
